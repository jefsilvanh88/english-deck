# Meu Deck — Inglês 🇬🇧→🇧🇷

Flashcards estilo Quizlet, PWA offline. Vira o card pra ver a tradução + exemplo.

**App:** https://jefsilvanh88.github.io/english-deck/

## Decks
- **Conectivos & Transições** — connectors / fillers (However, By the way, In a nutshell…)
- **Correções de Gramática** — erros comuns (taught, didn't see…)

## Usar
- Toca no card → vira EN↔PT
- Anterior / Sei ✓ / Próximo · Embaralhar · Resetar
- Teclado: ← → e espaço (virar)
- Progresso salvo no navegador. Funciona offline. Instalável (Adicionar à tela inicial).

## Adicionar palavras
Edita o bloco `const DECKS = [...]` no topo do `<script>` em [`index.html`](index.html) — marcado `EDITA AQUI`. Formato `{ en, pt, ex }`. Push pra `main` faz deploy automático.
